# Changesets

Este diretório gerencia versioning e releases dos packages do monorepo.

## Como usar

```bash
# Criar novo changeset após mudança nos packages
pnpm changeset

# Aplicar versões e atualizar CHANGELOGs
pnpm changeset version

# Ver status dos changesets pendentes
pnpm changeset status
```

> Ver ADR-0003 para a estratégia completa de versioning.
